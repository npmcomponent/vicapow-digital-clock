
# digital clock
 a simple retro digital clock
  

## Installation

    $ component install vicapow/digital-clock

## API

   var clock = new DigitalClock({seconds:true})
   clock.setHours(1)
   clock.setMinutes(23)
   clock.setSeconds(45)
   // or set via object
   clock.setTime(new Date)
